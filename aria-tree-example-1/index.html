ARIA Tree Example

This page provides an example of an ARIA tree that includes a variety of different ARIA tree functions. It does not provide any meaningful supoprt via the keyboard as this is a materially more complicated undertaking. Instead it focuses on providing a basic understanding of the updatdateable keyboard roles and ways to expose them.

Implementation Details

It is intended to demonstrate the following functions:

  • Set a label for the entire tree.
  • Set role="tree" for the overall tree container.
  • Set role="treeitem" for each tree item.
  • Set aria-expanded for expanded / collapsed state of nodes
  • Set the tree as a single tab stop with the first item receiving keyboard focus
  • Set each node of the tree as focusable but not in tab order (tabindex=-1)
  • Handle keyboard navigation throughout the tree and expand collapse
